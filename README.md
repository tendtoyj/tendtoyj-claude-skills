# tendtoyj-claude-skills

Claude Code skill plugins by [@tendtoyj](https://github.com/tendtoyj).

## Install

```
/plugin marketplace add tendtoyj/tendtoyj-claude-skills
```

## Plugins

### vibe-mkt

10 marketing skills — brand voice, positioning, copywriting, SEO, email, newsletters, and more.

```
/plugin install vibe-mkt@tendtoyj-claude-skills
```

| Skill | Command | Description |
|-------|---------|-------------|
| Orchestrator | `/vibe-mkt:orchestrator` | Marketing GPS. Diagnoses gaps and routes to the right skill |
| Brand Voice | `/vibe-mkt:brand-voice` | Extract and define your brand's voice and personality |
| Positioning Angles | `/vibe-mkt:positioning-angles` | Find differentiation using 8 positioning frameworks |
| Direct Response Copy | `/vibe-mkt:direct-response-copy` | Landing pages, ads, sales copy (Schwartz/Ogilvy/Halbert) |
| Lead Magnet | `/vibe-mkt:lead-magnet` | Design and build opt-in offers for lead capture |
| Email Sequences | `/vibe-mkt:email-sequences` | Welcome, nurture, conversion, and launch sequences |
| Keyword Research | `/vibe-mkt:keyword-research` | 6 Circles Method for SEO keyword discovery |
| SEO Content | `/vibe-mkt:seo-content` | Search-optimized, human-readable articles and guides |
| Newsletter | `/vibe-mkt:newsletter` | 9 newsletter formats for recurring engagement |
| Content Atomizer | `/vibe-mkt:content-atomizer` | Turn 1 piece into 15+ platform-specific assets |

### mkt-study

9 research skills — market scanning, competitive intelligence, customer profiling, and strategic synthesis.

```
/plugin install mkt-study@tendtoyj-claude-skills
```

| Skill | Command | Description |
|-------|---------|-------------|
| Research Orchestrator | `/mkt-study:research-orchestrator` | Research GPS. Diagnoses needs and builds optimal skill chain |
| Market Scanner | `/mkt-study:market-scanner` | Market size, structure, trends, and seasonality |
| Competitor Finder | `/mkt-study:competitor-finder` | Identify competitive set and map positioning |
| Competitor Analyzer | `/mkt-study:competitor-analyzer` | Deep-dive competitor websites: messaging, pricing, CTAs |
| Competitor Visual | `/mkt-study:competitor-visual` | Competitor design audit: screenshots, layout, visual tone |
| Audience Profiler | `/mkt-study:audience-profiler` | Customer segments, journey map, pain points |
| Voice of Customer | `/mkt-study:voice-of-customer` | Mine real customer language: pains, desires, triggers |
| Research Synthesizer | `/mkt-study:research-synthesizer` | Cross-analysis into actionable strategy brief |
| Expert Validator | `/mkt-study:expert-validator` | Multi-expert review of strategy brief |
