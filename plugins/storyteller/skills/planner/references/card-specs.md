## 템플릿 선택 가이드

| 템플릿 | 특징 | 추천 상황 |
|--------|------|-----------|
| **A (Full Image)** | 이미지가 카드 전체를 채움. 표지는 그라데이션으로 타이틀 강조. | 비주얼 임팩트 극대화, 사진/일러스트 중심 콘텐츠 |
| **B (Part Image)** | 이미지가 카드 일부를 차지. 텍스트 영역 분리. | 이미지 + 설명 텍스트 모두 중요한 콘텐츠, 가독성 우선 |

## 이미지 모드

모든 카드(표지 + 내용)에 이미지가 필요하다. 세트 전체에 하나의 모드를 통일 적용.

### Mode A: 시네마틱 빈티지
- 포토리얼리스틱, 사람 중심
- 극적인 자연광 (블라인드, 백라이트, 골든아워)
- 웜톤, 은은한 빈티지 필름 그레이딩
- 내러티브를 직접 재현하는 장면

### Mode B: 컨셉추얼 메타포
- 핵심 메타포를 오브젝트로 시각화
- 클린한 스튜디오 배경 (베이지/크림)
- 미니어처/디오라마, Wes Anderson 감성
- 에디토리얼 프로덕트 아트

### Mode C: 에디토리얼 미니멀
- 싱글 오브젝트 또는 미니멀 구도
- 밝고 에어리한 배경, 여백 활용
- 절제된 색감, 모던한 느낌

### Mode D: 초현실 아트
- 예상 밖의 스케일, 비현실적 조합
- 드림라이크 분위기, 풍부한 디테일
- 상상력이 필요한 주제에 적합

### Mode E: 다크 시네마틱
- 하이 콘트라스트, 깊은 그림자
- 누아르 분위기, 쿨톤
- 긴장감 있는 주제에 적합

### 이미지 주의사항

| 템플릿 + 카드 | 이미지 특성 |
|---------------|-------------|
| A — 표지 | 풀 이미지 + 그라데이션 오버레이 → 하단이 어두워짐, 상단에 주요 비주얼 |
| A — 내용 | 풀 이미지 배경 → 텍스트와 조화로운 비주얼, 너무 복잡하지 않게 |
| B — 표지 | 이미지 위 텍스트 오버레이 → 하단부 여백/어두운 영역 필요, 복잡하지 않은 구도 |
| B — 내용 | 상단 ~70%만 표시 → 독립적 비주얼, 하단 잘려도 무방 |
