## 카드 타입 선택 가이드

| 타입 | 이미지 필요 | 추천 상황 |
|------|-----------|-----------|
| **card-white** | 불필요 | 텍스트만으로 충분한 콘텐츠, 빠른 제작 |
| **card-image-overlay** | 필요 (풀이미지 + 텍스트 오버레이) | 비주얼 임팩트 강조, 짧은 카피 |
| **card-image-top** | 필요 (상단 이미지 + 하단 텍스트) | 이미지 + 설명 텍스트 모두 필요한 경우 |

## 이미지 모드

이미지가 필요한 카드 타입(card-image-overlay, card-image-top)에서만 사용.

### Mode A: 시네마틱 장면
- 포토리얼리스틱, 사람 중심
- 극적인 자연광 (블라인드, 백라이트, 골든아워)
- 웜톤, 은은한 빈티지 필름 그레이딩
- 내러티브를 직접 재현하는 장면

### Mode B: 컨셉추얼 메타포
- 핵심 메타포를 오브젝트로 시각화
- 클린한 스튜디오 배경 (베이지/크림)
- 미니어처/디오라마, Wes Anderson 감성
- 에디토리얼 프로덕트 아트

### 카드 타입별 이미지 주의사항

| 카드 타입 | 이미지 특성 |
|-----------|-------------|
| card-image-overlay | 텍스트가 위에 올라감 → 하단부 여백/어두운 영역 필요, 복잡하지 않은 구도 |
| card-image-top | 상단 ~70%만 표시 → 독립적 비주얼, 하단 잘려도 무방 |
