# TODAY'S PICKS 카드뉴스 템플릿 사용 가이드

## 개요

- **파일**: `card-news-template.html`
- **사이즈**: 1080 × 1350px (인스타그램 4:5)
- **구성**: 4장 (커버 → 이미지 콘텐츠 → 피처 콘텐츠 → 아웃로)
- **Placeholder 표기**: `{{placeholder-name}}` 형식

## 고정 요소 (수정 불필요)

다음은 디자인 구조 요소로, 콘텐츠 교체 시 건드리지 않아도 됩니다:

- `TODAY'S PICKS` 브랜드 헤더 + 도트 장식
- `In Brief` 라벨
- 상하단 구분선 (`.hdr__rule`, `.btm`)
- CSS 전체 (색상, 폰트, 레이아웃, 간격)
- 카드 셸 구조 (`.card`, `.card__inner`)

## 사용 팁

### 제목 하이라이트 구조
제목에서 노랑 하이라이트를 적용하려면 `<span class="hl">` 안에 키워드를 넣습니다:
```html
<span><span class="hl">키워드</span> 나머지 텍스트</span>
```

### 태그 개수 조절
커버의 태그 pill은 `<span class="pill">` 요소를 추가/삭제하여 2~4개로 조절할 수 있습니다.

### 피처 카드 아이콘
[Lucide Icons](https://lucide.dev/)에서 원하는 아이콘의 SVG path를 복사하여 `{{icon-N-path}}`에 넣으세요. path가 여러 개인 아이콘은 `<path>` 태그를 추가합니다.

### 이미지 사용
Card 2의 이미지 영역은 기본적으로 회색 placeholder입니다. 이미지를 넣으려면 주석을 해제하고 `src`를 교체합니다:
```html
<img src="실제-이미지-경로.jpg" alt="설명">
```

### 본문 줄바꿈
본문은 `<br>` 태그로 줄바꿈합니다. 3줄이 기본이지만 2~4줄로 조절 가능합니다 (단, 카드 높이가 고정이므로 넘침에 주의).
