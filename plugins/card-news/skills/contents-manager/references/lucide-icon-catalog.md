## Usage Notes

1. **Multi-path icons**: Some icons have multiple `<path>`, `<circle>`, `<rect>`, `<polyline>`, or `<line>` elements. Include ALL elements inside the `<svg>` tag.
2. **ViewBox**: Always use `viewBox="0 0 24 24"`.
3. **Styling**: The template CSS handles stroke styling via `.fcard__icon svg`. Do not add inline styles.
4. **Catalog updates**: This is a curated subset. For icons not listed here, visit [lucide.dev](https://lucide.dev/) and extract the SVG paths directly.
5. **Selection priority**: Prefer icons with fewer paths (cleaner at 32×32px rendering).
