## 2. Twitter / X

### Formats

**Thread** (primary for depth)
- Tweet count: 5-12
- Character limit per tweet: 280
- Structure:
  ```
  Tweet 1: [Hook â€” the entire pitch, works standalone] ðŸ§µ

  ```
- Tweet 1 is everything. If someone only reads one tweet, it should deliver value or create enough curiosity to pull them in.
- Each tweet must be retweetable on its own â€” no "And another thing..." that depends on the previous tweet for meaning.
- Use line breaks for emphasis within a tweet, not walls of text.
- Tone: concise, punchy, strong opening words. No fluff, no filler.

**Single Tweet**
- Character limit: 280
- Sweet spot: 240-270 (leave room for engagement)
- Use for: the sharpest Data or Bold Claim hook from your topic

- Pattern options:

### Hashtag Strategy

- Twitter/X rewards confidence and brevity. The same insight that takes 3 paragraphs on LinkedIn should take 2 sentences here.
